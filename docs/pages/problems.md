# 题目集

用户可在此处选择题目自己练习

## 题目列表

用于显示用户可自行练习的一些题目，每一道题目包含如下数据：

- 题目编号（整数）
- 题目名称（字符串）
- 题目难度
- 题目的算法标签
- 题目奖励的EMB
- 题目的通过人数
- 当前用户通过情况：没有进行任何提交（包括未登录），有过提交但是未通过，通过了

下面列出一个题目的json数据作为例子：

```json
{
    "id": 4951,
    "title": "查找整数",
    "reward": 2.87127520308282,
    "ac_count": 175,
    "knowledge_points": [
        {
            "id": 9,
            "name": "数组"
        },
        {
            "id": 22,
            "name": "二分查找"
        }
    ],
    "level": 1,
    "personal_label": -1
}
```

## 筛选器

用于根据题目难度和算法对题目进行筛选，也可以定性根据解出人数和奖励进行筛选
